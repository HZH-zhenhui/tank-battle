# 🎮 坦克大战 (Tank Battle)

一个使用纯 HTML5 Canvas + JavaScript 实现的经典坦克大战游戏。

## 🕹️ 操作方式

| 按键 | 功能 |
|------|------|
| `↑ ↓ ← →` 或 `W A S D` | 移动坦克 |
| `空格` 或 `J` | 发射炮弹 |
| `P` | 暂停 / 继续 |

## ✨ 特性

- 🎯 经典坦克大战玩法
- 🧱 可破坏的砖墙 & 不可破坏的钢铁墙
- 🤖 智能敌方 AI（追踪玩家方向射击）
- 💥 粒子爆炸特效
- 📈 关卡递增难度（敌人数量增加）
- ❤️ 生命值系统（3条命）
- 🛡️ 重生无敌时间
- 🎨 赛博朋克风格 UI
- 📱 响应式布局

## 🚀 运行方式

直接在浏览器中打开 `index.html` 即可开始游戏。

或者使用任意 HTTP 服务器：

```bash
npx serve .
# 或
python -m http.server 8000
```
